# FlutterWho
FlutterWho is a web-based butterfly classification tool powered by machine learning (MobileNetV2 Model). 
Upload an image via file or drag-and-drop to identify butterflies across 8 major categories with confidence scores, top predictions, and fascinating habitat facts.

# Features

- Classifies uploaded images (file upload or drag-and-drop) into any of 8 major butterfly categories.

- Displays model's top 3 predictions with confidence scores.

- Provides quick facts, habitat temperature range, and vegetation details for identified species.
